# Map Editor Application

Приложение для создания, редактирования и управления картами с объектами. Разработано на C# с использованием Windows Forms.

## Описание

Это десктопное приложение позволяет пользователям:
- Создавать и редактировать карты с помощью сетки
- Добавлять и размещать объекты на карте
- Сохранять и загружать карты
- Работать с объектами, включая обнаружение столкновений
- Интерпретировать и обрабатывать карты

## Системные требования

- **.NET 10.0** (Windows)
- **Windows Forms** для графического интерфейса
- **Операционная система**: Windows

## Ключевые компоненты

### Renderer
Отвечает за отрисовку сетки (grid) и объектов на карте. Поддерживает масштабирование и перемещение камеры.

### Map_info
Содержит логику работы с картой:
- Поиск объектов по названию
- Проверка столкновений между объектами
- Управление объектами карты

### MapSaver
Сохранение и загрузка данных карты из файловой системы.

### MouseHandle
Обработка пользовательского ввода через мышь для взаимодействия с объектами на карте.

### RedactorMode
Режим редактирования карты с возможностью добавления и редактирования объектов.

## Классы данных

### Mapobject
Базовый класс для всех объектов на карте:
- `position` - позиция объекта (Vector2)
- `size` - размер объекта (Vector2)
- `name` - название объекта

### Wall
Класс для стен (объекты для столкновений)

## Режимы работы

- **Режим редактора** (`redactormode`) - для создания и редактирования карт
- **Режим просмотра** - для просмотра готовой карты
